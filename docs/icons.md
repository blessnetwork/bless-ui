# Icons

All icon components are exported as React components with customizable `className`.

## Usage

```tsx
import {
	AirdropsIcon,
	DocumentationIcon,
	HomeIcon,
	LogoIcon,
	LogoutIcon,
	MenuIcon,
	MyNodesIcon,
	OrchestrationIcon,
	ReferralsIcon,
	SettingsIcon,
	StakingIcon
} from '@salsaflavio/bless-ui'

;<HomeIcon className="h-6 w-6 text-gray-600" />
```
